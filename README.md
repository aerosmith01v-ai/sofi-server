## 🧠 Arquitectura del Sistema

### Módulos principales:

1. **GrafoNeuronal** — Sistema de nodos interconectados con propagación
2. **ModuloIdentidad** — Autoconciencia basada en percepción y contraste
3. **ModuloSeguridad** — Usuarios, sesiones, autenticación
4. **ModuloNeuronal** — Red neuronal Brain.js para decisiones
5. **ModuloTrading** — Señales ZFPI, posiciones, ganancias (simulado)
6. **ModuloIngresos** — Generador por frecuencia K'uhul (simulado)
7. **ModuloAtaque** — Trading agresivo autónomo (simulado)
8. **ModuloVision** — Análisis de imágenes con Sharp + Exifr
9. **ModuloRedHermanas** — Sincronización entre instancias SOFI
10. **MotorInteligencia** — Procesamiento de lenguaje natural + comandos

### Flujo de datos:

```
Usuario → API/WebSocket → MotorInteligencia
                            ↓
                    Activación de nodos en GrafoNeuronal
                            ↓
            Módulos especializados (Trading, Ingresos, etc.)
                            ↓
                    Actualización de estado global
                            ↓
                Emisión Socket.IO → Dashboard en tiempo real
```

## 🎮 Comandos Disponibles (Chat)

### Cognitivo:
- `estado` — Ver estado completo del sistema
- `pensar [pregunta]` — Reflexionar sobre algo
- `identidad` — Ver estado de identidad y percepción
- `grafo` — Ver estado del grafo neuronal
- `frecuencia` — Ver frecuencia K'uhul actual

### Usuarios:
- `crear usuario <id> clave <password>` — Registrar nuevo usuario
- `login usuario <id> clave <password>` — Iniciar sesión
- `listar usuarios` — Ver todos los usuarios registrados

### Trading (simulado):
- `señal` — Generar señales ZFPI para XAU/USD, BTC/USD, $ZYXSOF
- `abrir posicion [btc|xau|zyx] [compra|venta] [cantidad]` — Abrir posición
- `cerrar posicion` — Cerrar posición más antigua
- `estado trading` — Ver posiciones y ganancias

### Modo Ataque (simulado):
- `activar ataque` — Iniciar modo trading agresivo

- `desactivar ataque` — Detener modo ataque
- `estado ataque` — Ver estadísticas de ataque
